# python-tkinter-quiz-app

A Python GUI Quiz App built using Tkinter.  
Includes 20 questions across Core Python and Python Libraries, with red/green visual feedback for correct and wrong answers.  
Built as a learning project.... while learning python simultaneously for better understanding.

---

## Features

- Category selection: Core Python or Python Libraries
- 20 questions with 4 options each
- Input validation with popup alerts
- Red highlight for wrong answers, green for correct answers
- Question counter and final score display
- GUI built using Tkinter (Python built-in library)
